Configuration Service

You can use the MdlService to turn off ripple effect & floating effect for the entire app. Just import the MdlService and change the values as bellow.


import {{'{'}} MdlService {{'}'}} from 'ng2-mdl';
export class App {{'{'}}
  ngOnInit() {{'{'}}
    MdlService.rippleEffect = false;
    MdlService.floating = false;
  {{'}'}}
{{'}'}}

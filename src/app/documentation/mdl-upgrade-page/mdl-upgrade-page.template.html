Upgrading Components

MDL Components inside *ngIf or dynamic templates needs to be manually upgraded as per the [MDL docs].
You can use the 'MdlUpgradeDirective' directive on elements that needs to be manually upgraded.

Follow this simple step:

Add the mdl, mdlUpgrade or mdl-upgrade attribute(directive) in the outer element of mdl components.


<label mdl class="mdl-switch mdl-js-switch mdl-js-ripple-effect" for="switch-1">
  <input type="checkbox" id="switch-1" class="mdl-switch__input" checked>
  <span class="mdl-switch__label"></span>
</label>

Done! :)

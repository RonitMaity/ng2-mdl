Progress

Uses:

  <md-progress-bar mode="indeterminate"></md-progress-bar>
      


  <md-progress-bar [value]="progress"></md-progress-bar>
      


  <md-progress-bar [value]="progress2" [bufferValue]="buffer"></md-progress-bar>
Supported Attributes
  • value: progress of the progress bar.
  • bufferValue: buffered progress of the progress bar.
  • mode: state of the progress. (default: determinate)

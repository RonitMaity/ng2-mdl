Select

Material Design Lite doesn't have its own select component. You'll need to install a separate library for this. Thanks to mdl-selectfield.


npm install --save mdl-select-component
Uses:

Component:
binding: string = 'value3';
View:
<mdlSelect [(ngModel)]="binding"> <!-- or [value]="binding" or value="value3" -->
  <option value="value1">Option 1</option>
  <option value="value2">Option 2</option>
  <option value="value3">Option 3</option>
</mdlSelect>
{{binding}}
Supported Attributes
  • id: id of the select element.
  • class: class of the select element.
  • value: value of the select element.
  • label: text of the label element.
  • disabled: disable select. (default: false)

Switch

Uses:

  <mdlSwitch label="Switch" [(ngModel)]="example"></mdlSwitch>

{{example ? "Switch On" : "Switch Off"}}

Supported Attributes

This component comes with supports for angular2 ngModel & ngFormControl directives. Here are some attributes you can use to pass data to the component.

  • class: class of the input element.
  • value: value of the input element. (default: true)
  • label: text of the label element.
  • ripple: ripple effect for the checkbox. (default: true)

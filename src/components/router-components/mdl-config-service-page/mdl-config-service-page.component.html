Configuration Service

You can use the MdlConfigService to turn off ripple effect & floating effect for the entire app. Just import the MdlConfigService and change the values as bellow.


import { MdlConfigService } from 'ng2-mdl';
export class App {
  ngOnInit() {
    MdlConfigService.rippleEffect = false;
    MdlConfigService.floating = false;
  }
}

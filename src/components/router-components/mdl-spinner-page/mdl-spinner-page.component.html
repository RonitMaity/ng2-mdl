Spinner

Import & Inject

import {MdlSpinnerComponent} from 'ng2-mdl/ng2-mdl';
@Component({
  ...
  directives: [MdlSpinnerComponent],
  ...
})
Uses:

  <mdlSpinner></mdlSpinner>
  <mdlSpinner [singleColor]="true"></mdlSpinner>
  <mdlSpinner [active]="false"></mdlSpinner>
Supported Attributes
  • spinner: spinner of the spinner bar.
  • buffer: buffered spinner of the spinner bar.
  • indeterminate: indeterminate state. (default: false)

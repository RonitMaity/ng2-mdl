Text Inputs

Import & Inject

import {MdlTextFieldComponent} from 'ng2-mdl/ng2-mdl';
@Component({
  ...
  directives: [MdlTextFieldComponent],
  ...
})
Uses:

  <mdlTextField label="eMail" type="email" errMsg="Invalid email!" [(ngModel)]="email"></mdlTextField>
  or
  <mdl-text-field label="eMail" type="email" errMsg="Invalid email!" [(ngModel)]="email"></mdl-text-field>
{{ email }}
Supported Attributes
This component comes with supports for angular2 ngModel & ngFormControl directives. Here are some attributes you can use to pass data to the component.
  • id: id of the input element.
  • class: class of the input element.
  • type: type of the input element. (default: text)
  • value: value of the input element.
  • label: text of the label element.
  • floating: floating label. (default: true)
  • errMsg: error message to display when HTML5 form validation fails.

Progress

Import & Inject

import {MdlProgressComponent} from 'ng2-mdl/ng2-mdl';
@Component({
  ...
  directives: [MdlProgressComponent],
  ...
})
Uses:

  <mdlProgress [indeterminate]="true"></mdlProgress>
  <mdlProgress [progress]="progress"></mdlProgress>
  <mdlProgress [progress]="progress2" [buffer]="buffer"></mdlProgress>


Supported Attributes
  • progress: progress of the progress bar.
  • buffer: buffered progress of the progress bar.
  • indeterminate: indeterminate state. (default: false)

Text Inputs

Import & Inject

import {MdlCheckBoxComponent} from 'ng2-mdl/ng2-mdl';
@Component({
  ...
  directives: [MdlCheckBoxComponent],
  ...
})
Uses:

  <mdlCheckBox label="Are you ready?" [(ngModel)]="example"></mdlCheckBox>
  or
  <mdl-check-box label="Are you ready?" [(ngModel)]="example"></mdl-check-box>

{{ready ? "Yes! I am." : "Not Yet!"}}

Supported Attributes
This component comes with supports for angular2 ngModel & ngFormControl directives. Here are some attributes you can use to pass data to the component.
  • class: class of the input element.
  • value: value of the input element. (default: false)
  • label: text of the label element.
  • ripple: ripple effect for the checkbox. (default: true)

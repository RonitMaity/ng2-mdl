Radio Buttons

Import & Inject

import {MdlRadioComponent} from 'ng2-mdl/ng2-mdl';
@Component({
  ...
  directives: [MdlRadioComponent],
  ...
})
Uses:

  <mdlRadio label="Male" value="male" [(ngModel)]="gender"></mdlRadio>
  or
  <mdl-radio label="Female" value="female" [(ngModel)]="gender"></mdl-radio>

Gender:

{{gender === 'male' ? "Male" : "Female"}}

Supported Attributes
This component comes with supports for angular2 ngModel & ngFormControl directives. Here are some attributes you can use to pass data to the component.
  • class: class of the input element.
  • value: value of the input element.
  • label: text of the label element.
  • ripple: ripple effect for the checkbox. (default: true)
